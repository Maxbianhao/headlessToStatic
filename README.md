# headlessToStatic
通过chrome headless生成静态页

## 指令
	
	# 启动抓取
	npm run dev
	
## 参数

参数|说明
---|---
--all|全部页面
--index|首页
--startdesign|开启设计页面
--templatecenter-f|模版中心一级页面
--templatecenter-s|模版中心二级页面
--fid-1|模版中心二级页面一级分类id（必须使用templatecenter-s指令）
--sid-11|模版中心二级页面二级分类id（必须使用templatecenter-s跟-fid指令）
--templatecenter-m|模版中心模版详情页
